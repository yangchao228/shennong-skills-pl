# Skills Manager Alpha Trial Feedback

Use this template after trying `v0.1.0-alpha` with a small local skills
directory.

Recommended trial scope:

- Copy 2-3 real skills into a temporary directory.
- Point `SKILLS_PATH` at that temporary directory.
- Try one verifiable skill end to end: inspect, generate tests, run tests, save
  baseline, modify `SKILL.md`, inspect diff, restore or apply a candidate.
